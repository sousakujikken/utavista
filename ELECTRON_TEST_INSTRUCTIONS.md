# Electronアプリでの座標確認手順

現在、Electronアプリが起動しているので、手動で以下の手順を実行してください：

## 手順

### 1. 基本設定
1. Electronアプリで「テンプレート」タブを開く
2. テキストエリアに以下の歌詞を入力：
```
テストABC
あいうえお
12345
```

### 3. テンプレート選択
1. テンプレート選択で「Primitive WordSlideText」を選択
2. パラメータを以下に設定：
   - fontSize: 80
   - charSpacing: 1.5
   - fontFamily: 適当なフォント

### 4. タイミング生成
1. 「自動生成」ボタンをクリックしてタイミングデータを生成

### 5. 再生と座標確認
1. 再生ボタン（▶）をクリック
2. 開発者ツール（F12またはCmd+Option+I）を開く
3. コンソールタブで以下のログを確認：
   - `[PrimitiveWordSlideText] Char X "Y":` - 各文字の配置情報
   - `📍 VISIBLE CHAR "Y":` - 表示中の文字の座標
   - `After animation - Char "Y" final position:` - 最終位置

### 6. 期待される結果
正しく修正されていれば：
- 各文字のlocalXが異なる値（0, 80, 160, 240...など）
- worldXも異なる値で水平方向に並ぶ
- 文字が重ならずに水平に配置される

### 7. 問題がある場合
文字が重なっている場合：
- localXがすべて0になっている
- worldXがすべて同じ値になっている
- 座標ログが出力されない

## 結果をコンソールログで確認し、座標の値を報告してください。
